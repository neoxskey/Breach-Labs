## Packages
framer-motion | Animations for UI transitions, modals, and progress bars
recharts | Visualizing user progress and stats on the dashboard
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes efficiently

## Notes
The application requires a dark, cyberpunk aesthetic.
Authentication is handled via custom API endpoints (/api/login, /api/register).
Lab simulation logic is client-side but syncs progress to backend via /api/progress.
Lucide icons are used extensively for the UI.
